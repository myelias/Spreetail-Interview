MultiValueDict Usage Guide

Overview

This project implements a MultiValueDict data structure and provides a test suite to validate its functionality.

Prerequisites:

-Python 3.9.6
-No external dependencies required (uses Python standard library)

Running the MultiValueDict Program

To execute the MultiValueDict program interactively, use the following command:

python3 multiValueDict.py

This will launch an interactive prompt where you can enter commands to interact with the dictionary.

Running the Test Suite

To run the unit tests, use:

python3 -m unittest multiValueDictTest.py

This will execute all test cases and display the results.

File Descriptions

multiValueDict.py - The main implementation of the MultiValueDict.

multiValueDictTest.py - A test suite using unittest to validate the functionality of MultiValueDict.

README.md - This file, providing instructions on how to run the program and tests.

Notes

Ensure that both multiValueDict.py and multiValueDictTest.py are in the same directory when running tests.

Use Python 3.9.6 for compatibility with the provided code.
